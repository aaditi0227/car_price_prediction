# 🚗 Car Price Prediction using Streamlit

A Machine Learning web application that predicts the price of a car based on user inputs such as mileage, fuel type, transmission, etc.

---

## 📌 Project Overview

This project uses a trained ML model to estimate car prices. Users can input car details through a Streamlit web interface and get instant predictions.

---

## 🧠 Machine Learning Model

- Algorithm: Random Forest (or change based on your model)
- Libraries Used:
  - scikit-learn
  - pandas
  - numpy

---

## 🖥️ Tech Stack

- Frontend: Streamlit  
- Backend: Python  
- ML: Scikit-learn
